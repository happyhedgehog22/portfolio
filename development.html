Background

This project was done for CS1300: UI/UX. The goal of the assignment was to build a website using React to create a list-based interface

I chose to design an interface that will allow users to browse movies and add them to a list to watch. I wanted to make an interface that I would use and I frequently find myself scrolling through streaming platforms looking for movies to watch.

I had very little React experience before this project, so it was a bit difficult at the beginning to figure out how to create Components and states. I had to learn general architecture, Components, state, and props.

Project Design

Ideation

I began by coming up with a list of all the features I wanted

  • Filter by genre & length
  • Sort by rating
  • Ability to add and remove from to watch list

Coding

The next step was figuring out how to structure this in code

I ended up creating two Components: one for movies and one for the to-watch list. The movie Component takes in props which is a movie from movie-data.json and displays the movies' name, rating, length, genre, and image. It also has buttons to add or remove from the aggregator. The Component for the aggregator displays the movies added to your list and the total time of all the movies on your list.

Result

The website allows you to filter by genre, length, and sort by rating. It also allows you to add and remove movies to your list and tells you the total time of all the movies on your list.

Conclusion

This project taught me a number of skills. I learned how to make a webapp using React. I learned how to model items using Components, use props to pass data to Components, and useState to update values.
